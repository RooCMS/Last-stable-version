Cистемные требования
--------------------
 - Сервер:	`*nix Apache`
 - PHP:		`5.2.*` `5.3.*` `5.4.*` `5.5.*`
 - MySQL:	`5.*`
 - PHP Extension: 
	`Core`
	`calendar`
	`date`
	`pcre`
	`session`
	`xml`
	`gd`
	`mbstring`
	`standard`
	`SimpleXML`
	`apache2handler`
	`mysqli`

Установка
---------
Для установки RooCMS скачайте последнию сборку платформы с сайта <http://www.roocms.com>. Распакуйте архив на свой хостинг и через ваш браузер зайдите по адресу: `http://www.your_site.ru/install/`

Wiki
----
Помошь и подсказки по работе с RooCMS, находятся в панели управления. 
После установки, зайдите в Панель Управления - "Помощь".
